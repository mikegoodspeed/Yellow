# Project
Small Pygame prototypes (menu + game + snake + tutorials).

# Setup
- `python -m venv .venv`
- `source .venv/bin/activate`
- `python -m pip install -U pip`
- `python -m pip install pygame`

# Run
- Main menu/game bundle: `python mellow_yellow.py`
- Tutorial flow: `python Yellow.py`
- Snake standalone: `python snake.py`

# Tests
- `python -m pytest -q`
